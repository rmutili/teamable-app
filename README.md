# teamable-app
Teamable app using vue
Install Node Js
Install Vue dependences in a package.json file
      {
        "name": "teamable",
        "version": "1.0.0",
        "dependencies": {
              "vue": "3.2.40",
              "core-js": "3.25.5",
              "@vue/compiler-sfc": "3.2.40",
              "@vue/cli-service": "5.0.8"
            },
            "scripts": {
              "serve": "vue-cli-service serve"
            }
          }
==> npm i /* This installs all dependences */

run application
  npm run serve
